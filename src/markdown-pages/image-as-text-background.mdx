Using an image to visually fill text

```css
@import url("https://fonts.googleapis.com/css?family=Monoton&display=swap");

body {
  height: 100vh;
  display: flex;
  justify-content: center;
  align-items: center;
  font-family: "Monoton", cursive;
}

h1 {
  font-size: 15vw;
  background-image: url(https://images.unsplash.com/photo-1505143050064-819ec023399f);
  background-size: cover;
  background-clip: text;
  -webkit-background-clip: text;
  color: transparent;
}
```

```css
@import url("https://fonts.googleapis.com/css?family=Monoton&display=swap");

body {
  height: 100vh;
  display: flex;
  justify-content: center;
  align-items: center;
  font-family: "Monoton", cursive;
  background: linear-gradient(
      to left,
      rgba(200, 090, 090, 0.3),
      rgba(005, 0, 0, 0.1),
      rgba(090, 090, 200, 0.3)
    ), url(https://images.unsplash.com/photo-1524207874394-5ec7c8c8e1a6);
  background-size: cover;
}

h1 {
  font-size: 20vw;
  background-image: url(https://images.unsplash.com/photo-1505143050064-819ec023399f);
  background-size: cover;
  background-clip: text;
  -webkit-background-clip: text;
  color: rgba(0, 0, 0, 0.2);
  text-align: center;
}
```
